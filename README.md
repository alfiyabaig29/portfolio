# Alfiya Baig — Personal Portfolio

A fully responsive personal portfolio website built with HTML, CSS, and JavaScript as part of an internship task. Showcases my skills, projects, work experience, and research in a clean, elegant design.


## File Structure

PORTFOLIO/
├── index.html       ← Main HTML structure and content
├── style.css        ← All styles and theme variables
├── script.js        ← Cursor, scroll, and reveal animations
└── README.md        ← Project documentation


## Built With

- **HTML5** — Semantic structure and content
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript** — Scroll-based nav, reveal animations, custom cursor
- **Google Fonts** — Cormorant Garamond, Jost, DM Mono


## Features

- Full-screen cinematic landing page
- Smooth scroll-reveal animations
- Fixed navigation bar that appears on scroll
- Custom animated cursor
- Responsive design (mobile & desktop)
- Work experience timeline
- Bento-grid skills section
- Project showcase cards
- Research / academic paper section
- Contact form with mailto integration

##  How to Run Locally

1. Open the project folder in VS Code
2. Make sure the **Live Server** extension is installed
3. Right-click on `index.html` and select **Open with Live Server**
4. The project will open in your browser at `http://127.0.0.1:5500`

> Ensure `index.html`, `style.css`, and `script.js` are in the same folder.
